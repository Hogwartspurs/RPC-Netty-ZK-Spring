# 项目
- 基于Netty、Zookeeper、Spring的轻量级、高性能的RPC框架
## 特征

- 支持长连接
- 支持异步调用
- 支持心跳检测
- 支持JSON序列化
- 接近零配置，基于注解实现调用
- 基于Zookeeper实现服务注册中心
- 支持客户端连接动态管理
- 支持客户端服务监听、发现功能
- 支持服务端服务注册功能
- 基于Netty4.X版本实现
